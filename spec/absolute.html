Absolute Objects

An `Absolute` is an immutable object referencing a point in time with nanoseconds precision.

new Absolute(*nanoseconds*)

When called as a constructor creates a new Absolute object. *nanoseconds* is a BigInt of the nanoseconds since epoch.

Absolute.prototype.epochSeconds

`absolute.epochSeconds` is a getter that returns the seconds Number integer since epoch of the point in time represented by the absolute object such that `absolute.epochSeconds === Number(absolute.valueOf() / BigInt(1e9)` is *true*.

Absolute.prototype.epochMilliseconds

`absolute.epochMilliseconds` is a getter that returns the milliseconds Number integer since epoch of the point in time represented by the absolute object such that `absolute.epochMilliseconds === Number(absolute.valueOf() / BigInt(1e6))` is *true*.

Absolute.prototype.epochMicroseconds

`absolute.epochMicroseconds` is a getter that returns the microseconds BigInt since epoch of the point in time represented by the absolute object such that `absolute.epochMicroseconds === Number(absolute.valueOf() / BigInt(1e3))` is *true*.

Absolute.prototype.epochNanoseconds

`absolute.epochNanoseconds` is a getter that returns the nanoseconds BigInt since epoch of the point in time represented by the absolute object such that `absolute.epochNanoseconds === absolute.valueOf()` is *true*.

Absolute.prototype.withZone(_timezone_)

When called with an Absolute object as a _this_ context it will return a ZonedDateTime object that specifies an absolute in time in the given time zone.

_timezone_ is a string specifying a time zone. Legal values are IANA time zone names.

Calling this function with a _this_ context that is not an instance of Absolute is illegal.

Absolute.prototype.toString()

When called with an Absolute object as a _this_ context it will return an ISO 8601 String value representing the value of the Absolute object.

The format of the String will always be an IsoDateTime of the following form:

Calling this function with a _this_ context that is not an instance of Absolute is illegal.

Absolute.fromString(_string_)

This function will create an Absolute object from a String. The _string_ has to strictly conform to the format produced by . Any _string_ not strictly conforming to that format will throw an Error.

Absolute.fromEpochSeconds(_seconds_)

This function will create an Absolute object that represents the absolute in time _seconds_ since epoch such that `BigInt(absolute.epochSeconds) * BigInt(1e9) === absolute.epochNanoseconds` and `absolute.epochSeconds === _seconds_` are both `true`.

Absolute.fromEpochMilliseconds(_milliseconds_)

This function will create an Absolute object that represents the absolute in time _milliseconds_ since epoch such that `BigInt(absolute.epochMilliseconds) * BigInt(1e6) === absolute.epochNanoseconds` and `absolute.epochMilliseconds === _milliseconds_` are both `true`.

Absolute.fromEpochMicroseconds(_microseconds_)

This function will create an Absolute object that represents the absolute in time _milliseconds_ since epoch such that `absolute.epochMicroseconds * BigInt(1e3) === absolute.epochNanoseconds` and `absolute.epochMicroseconds === _microseconds_` are both `true`.

Absolute.fromEpochNanoseconds(_nanoseconds_)

This function will create an Absolute object that represents the absolute in time _nanoseconds_ since epoch such that `absolute.epochNanoseconds === absolute.epochNanoseconds` and `absolute.epochMicroseconds === _microseconds_` are both `true`.
